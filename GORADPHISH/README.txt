

 GoradPhish (Tools phishing for penetration testing ) OPEN SOURCE TOOLS 

- GoradPhish  est un outil de phishing conçu pour simuler des attaques de phishing sur Facebook, dans le but de sensibiliser à la sécurité et de former des utilisateurs ou des administrateurs à reconnaître ces menaces. **Attention** : Cet outil doit être utilisé de manière éthique, uniquement dans des environnements autorisés pour des tests de sécurité.

Auteur *

Pseudo : Cybergorad



- Description
GoradPhish permet de créer des pages de phishing Facebook réalistes qui peuvent être utilisées pour démontrer et expliquer les risques des attaques de phishing. L'outil est simple à utiliser, avec des options permettant de personnaliser l'attaque pour mieux comprendre la sécurité des réseaux sociaux.

> DISCLAIMER : L'utilisation non autorisée de cet outil pour des activités malveillantes est strictement interdite. Assurez-vous de respecter les lois en vigueur et d'avoir l'autorisation de l'organisation cible avant d'exécuter des simulations de phishing.



- Fonctionnalités
- Création de pages de phishing Facebook personnalisées.
- Prise en charge des tests de sensibilisation à la sécurité.
- Journalisation des informations soumises (pour des fins de démonstration et de formation).
- Interface simple pour facilité d'utilisation 


     Prérequis
- Termux ou un environnement Linux installé sur machine ou PC
- Python3 ou Python installé.
- MySQL Connector: pour Python (installé via `pip`).
- Connexion Internet stable . 




 Installation
1. Clonez le dépôt :
   git clone https://github.com/cybergorad/GoradPhish.git


2. Accédez au dossier de l'outil :

   cd GoradPhish
   
3. Installez les dépendances requises :

   pip install -r requirements.txt
 
   
## Utilisation
1. Lancez l'outil avec Python :
  
   python3 goradphish.py

2. Suivez les instructions pour configurer l'attaque de phishing.
3. Les résultats des soumissions seront enregistrés pour analyse (uniquement à des fins de sensibilisation ou de formation).

 Avertissement 
L'utilisation de GoradPhish doit être éthique et respecter les lois. Utilisez cet outil uniquement pour des tests de sécurité autorisés. Toute utilisation abusive est strictement interdite.
